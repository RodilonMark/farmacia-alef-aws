# 🏥 Farmácia Alef - Projeto AWS

Este repositório apresenta a implementação de três serviços da **Amazon Web Services (AWS)** na Farmácia Alef, com foco em **redução de custos** e **aumento de eficiência**.

## 📂 Arquivos do Projeto
- [Relatório de Implementação](relatorio.md)
- [Manual Amazon S3](manual-s3.md)
- [Guia Amazon RDS](guia-rds.md)
- [Scripts AWS Lambda](scripts-lambda.md)

## 🎯 Objetivo
Demonstrar como soluções em nuvem podem otimizar processos internos e reduzir gastos em uma farmácia fictícia.

## 🛠️ Tecnologias Utilizadas
- **Amazon S3** → Armazenamento seguro e escalável em nuvem.
- **Amazon RDS** → Banco de dados relacional gerenciado.
- **AWS Lambda** → Execução de código sob demanda, sem servidores.
