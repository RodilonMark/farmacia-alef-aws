# Guia Amazon RDS

O Amazon RDS é um serviço de banco de dados gerenciado.  
Ele facilita a criação, operação e escalabilidade de bancos de dados relacionais.  

- **Benefício:** Redução de custos com manutenção de servidores.  
- **Uso na Farmácia Alef:** Gerenciar estoque e vendas de forma confiável.
