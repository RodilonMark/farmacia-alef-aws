# Scripts AWS Lambda

O AWS Lambda permite executar código sem precisar de servidores dedicados.  
Ele é ideal para automação de tarefas rápidas e sob demanda.  

- **Benefício:** Redução de custos e maior agilidade.  
- **Uso na Farmácia Alef:** Enviar notificações de promoções e lembretes de medicamentos para clientes.
